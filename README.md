# Exploratory Data Analysis (EDA) on Titanic Dataset

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic Dataset to understand the data using descriptive statistics, visualizations, and pattern recognition techniques.

## Dataset

Titanic Dataset

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly

## Tasks Performed

### Dataset Exploration

* Loaded and explored the Titanic dataset.
* Examined dataset dimensions and feature information.

### Missing Value Analysis

* Identified missing values in Age, Cabin, and Embarked columns.

### Summary Statistics

* Generated statistical measures including:

  * Mean
  * Median
  * Standard Deviation
  * Minimum
  * Maximum

### Data Visualization

* Histograms
* Boxplots
* Correlation Heatmap
* Pairplot Analysis
* Survival Distribution Analysis
* Gender-wise Survival Analysis
* Passenger Class Analysis
* Fare Analysis

### Pattern Recognition

* Identified trends and relationships between features.
* Detected outliers and anomalies.
